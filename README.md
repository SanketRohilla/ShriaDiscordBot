# Shria – AI Discord Bot 🤖💖

Hey! This is my personal Discord bot project called **Shria**.

It’s a mix of fun + useful features:
- 💬 Chat with an AI personality
- 🎬 Send anime reaction GIFs
- 🎵 Play music in voice channels

I built this while learning and experimenting with Discord bots, APIs, and hosting.

---

## ✨ Features

### 💬 AI Chat
Talk to Shria like a real person. She gives short, playful replies.

### 🎬 Anime Actions
Commands like:
- kiss  
- hug  
- slap  
- blush  

These return anime GIFs automatically.

### 🎵 Music System
Play songs directly in voice channels:

- `/play <song>` – play music  
- `/skip` – skip current track  
- `/pause` – pause/resume  
- `/stop` – stop music  
- `/leave` – leave voice channel  

---

## ⚙️ Setup

1. Clone or download this repo

2. Install dependencies:

pip install -r requirements.txt


3. Add environment variables:
- TOKEN (your Discord bot token)
- GROQ_API_KEY
- GIPHY_API_KEY

4. Run the bot:

python main.py


---

## ☁️ Hosting

If you want it running 24/7, I recommend using Railway.

Replit is fine for testing, but not reliable for voice/music features.
